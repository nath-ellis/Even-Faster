# Even Faster
Even Faster is a game about dodging cars driving towards you.

## Controls
Up Arrow / W to go up \
Down Arrow / S to go down \
Left Arrow / A to go left \
Right Arrow / D to go right \
F1 to toggle Siren \
F2 to toggle music
